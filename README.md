


# Customer Churn Prediction

## Overview
This project predicts customer churn using machine learning techniques. The dataset is preprocessed, encoded, and used to train a Random Forest classifier to identify customers who are likely to leave a company's service.

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Machine Learning Workflow
- Data Cleaning
- Data Preprocessing
- One-Hot Encoding
- Train-Test Split
- Random Forest Classification
- Model Evaluation
- Confusion Matrix
- ROC Curve
- Feature Importance

## Dataset
Telco Customer Churn Dataset

## Results
The model was evaluated using Accuracy, Precision, Recall, F1-Score, Confusion Matrix, and ROC-AUC.
